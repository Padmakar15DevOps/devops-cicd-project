# CI/CD Pipeline using GitHub, Jenkins and Docker

## Project Overview

This project demonstrates a complete CI/CD pipeline using GitHub, Jenkins and Docker.

## Technologies Used

* GitHub
* Jenkins
* Docker
* Nginx

## Pipeline Flow

GitHub → Jenkins → Docker Build → Docker Container Deployment

## Features

* Automated build process
* Automated Docker image creation
* Automated container deployment
* Continuous Integration and Continuous Deployment
